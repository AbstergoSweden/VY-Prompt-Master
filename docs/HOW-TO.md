# How-To

This document provides step-by-step instructions for using the VY Prompt Master repository.

## Setup Repository
- Clone the repository using `git clone https://github.com/AbstergoSweden/VY-Prompt-Master.git`.
- Install dependencies if any (none for this documentation repository).

## Create a New Persona
- Navigate to the `personas/` directory.
- Copy an existing YAML persona file and modify fields or create a new file following the same structure.

## Add a Response YAML
- Store response YAML files in the `responses/` directory.
- Follow the response schema defined in the repository.

## Update Documentation
- Place new documentation files in the `docs/` folder.
- Update the README to reflect any changes to structure or usage.
